# Human Development Index (HDI) Prediction System

## Project Description
This project predicts the Human Development Index (HDI) based on three input values:
- Life Expectancy Index
- Education Index
- Income Index

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install the required libraries:
   pip install -r requirements.txt

2. Run the project:
   python app.py

3. Open your browser and visit:
   http://127.0.0.1:5000
